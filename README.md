## 🖥️ What bfjit is

bfjit is a small Windows app that runs Brainfuck code with a JIT compiler. That means it turns code into machine work fast, so it can run with less delay.

Use it if you want to:

- Run Brainfuck programs on Windows
- Test small code samples
- Work with a fast local tool
- Keep setup simple
